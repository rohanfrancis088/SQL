E-Commerce Sales Analysis using SQL

Overview -: This project analyzes an e-commerce dataset to extract meaningful business insights using SQL. The objective is to understand customer behavior, sales performance, and operational efficiency to support data-driven decision-making.

Objectives -:

Analyze revenue trends over time
Identify top-performing products and categories
Segment customers based on spending behavior
Evaluate city-wise sales performance
Analyze payment methods
Calculate key KPIs like Average Order Value and Costumer Life Time Value
Database Schema -:

create table salesdata( order_id int primary key, customer_id varchar(50), order_date varchar(50), product_category varchar(50), Product_name varchar(50), Quantity int, Unit_Price int, Total_Price int, Payment_Type varchar(50), Delivery_Days int, Customer_City varchar(50) );
